# Quick Reference: Routine Compliance & Content Production KPIs

## 🎯 Routine Compliance KPI

**What it does**: Tracks adherence to a daily/weekly routine and shows how it impacts a physical outcome.

**Real-world example**:
```
Goal: "Reduce body fat to 20%"
├─ Routine: "Gym 3x/week"
├─ KPI: "Maintain 80% gym adherence over 100 days"
└─ Result: "Body fat: 22% → 20%"
```

### How to Use

1. **Create a Routine** (in Routines section):
   - Name: "Gym 3x/Week" or "Meal Prep Sunday"
   - Frequency: 3 times per week
   - Link to Goal: "Get fit"

2. **Create Routine Compliance KPI**:
   - Type: "Routine Compliance"
   - Name: "Gym Adherence 80% (100 days)"
   - Target: 80 (%)
   - Lookback: 100 (days)
   - Link Routine: "Gym 3x/Week"
   - Link Outcome: "Body Fat %" (current: 22%, target: 20%)

3. **Mark Routine Complete**:
   - Check off routine in daily checklist
   - System auto-calculates adherence %
   - Shows: "82 / 100 days = 82%" ✓

4. **Monitor Progress**:
   - Dashboard shows routine adherence trending
   - Manual entries update linked metric (body fat %)
   - Both visible in Focus Goals countdown banner

### Status Calculation

| Adherence | Target | Status |
|-----------|--------|--------|
| 82% | 80% | ✓ On Target (102%) |
| 78% | 80% | ↗ OK (97%) |
| 65% | 80% | ⚠ Behind (81%) |

## ✍️ Content Production KPI

**What it does**: Tracks consistent content creation and publishing across any platform.

**Real-world example**:
```
Goal: "Build transformation journey audience"
├─ Platform: Substack
├─ KPI: "Write 2 articles/week"
└─ Result: "4 articles published | 8.5/10 quality | 3 drafts"
```

### How to Use

1. **Create Content Tasks**:
   - Title: "Write Substack: Gym Progress"
   - Title: "Write Substack: Nutrition Update"
   - Mark task done when published

2. **Create Content Production KPI**:
   - Type: "Content Production"
   - Name: "Substack 2x/Week"
   - Target: 2 (articles)
   - Content Type: "article"
   - Platform: "substack"
   - Link Tasks: (select the tasks above)

3. **Publish Content**:
   - Complete task when article goes live
   - System counts items published this period
   - Shows: "4 articles this week = 200% ✓"

4. **Track Quality**:
   - Optional: Set quality score (1-10)
   - Track articles in draft/backlog
   - Monitor last published date

### Progress Calculation

| Published | Target | Progress | Status |
|-----------|--------|----------|--------|
| 4 | 2 | 200% | ✓ On Target (exceeding!) |
| 2 | 2 | 100% | ✓ On Target |
| 1 | 2 | 50% | ⚠ Behind |

## 💡 Tips & Best Practices

### For Routine Compliance

1. **Set realistic targets**: 80-90% is achievable; 100% leads to burnout
2. **Link visible outcomes**: Show the physical result (body fat %, weight, mood)
3. **Use consistent lookback**: 30 days (monthly), 100 days (quarterly), 365 days (annual)
4. **Celebrate consistency**: When you hit target, increase difficulty or add new routine

### For Content Production

1. **Batch your content**: Write 4 articles at once, publish over 4 weeks
2. **Track quality**: Don't just count; rate your best work 8+/10
3. **Maintain backlog**: Keep 3-5 drafts ready to publish
4. **Repurpose content**: One blog post → LinkedIn post → Email newsletter → Tweet thread

## ❓ FAQ

**Q: How is routine adherence calculated?**
A: (Days completed / Total days in period) × 100. Example: 82 days / 100 days = 82%

**Q: Can I link multiple routines to one KPI?**
A: Yes! You can specify multiple routine IDs. System counts all of them.

**Q: How are content items counted?**
A: By counting completed tasks linked to the KPI. Complete task = 1 item published.

**Q: Can I manually adjust routine/content KPI values?**
A: Yes, for content (edit task status). For routines, system auto-calculates from completed dates.

**Q: What if I miss a day of my routine?**
A: Your adherence % goes down slightly. Example: 81 days / 100 days = 81%. Still visible on dashboard.

**Q: Can I have multiple KPIs of the same type per goal?**
A: Yes! Example: Gym routine AND meal prep routine, both driving body fat goal.
